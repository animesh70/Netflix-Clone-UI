# Netflix Clone UI

A responsive Netflix-inspired landing page built using HTML and CSS. This project was created to practice frontend development, responsive web design, and modern UI styling techniques.

## Features
- Responsive design for different screen sizes
- Netflix-inspired modern user interface
- Flexbox and CSS positioning
- Clean layout and structured sections
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3

## What I Learned
- Responsive web development
- UI layout structuring
- Flexbox positioning
- Modern webpage styling techniques
- Improving website responsiveness across devices

## Project Status
Completed frontend UI clone project for learning and practice purposes.
